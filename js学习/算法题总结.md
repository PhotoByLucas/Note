## 使一个数反转 如 123->321
  + slice()  
    slice 方法可以用来将一个类数组（Array-like）对象/集合转换成一个新数组。
    ~~~
    function list() {
      return Array.prototype.slice.call(arguments);
      // 也可以使用[].slice.call代替
    }

    var list1 = list(1, 2, 3); // [1, 2, 3]
    ~~~
  + str.split('') 将字符串切割为数组
  + array.reverse()  反序
  + array.join('') 将数组拼接为字符串
	~~~
	num.toString().split('').reverse().join('')
	~~~
