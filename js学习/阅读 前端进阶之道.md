## typeof
  + 对于除了null意外都显示为 正确类型
    ~~~
    typeof null // 'object' 
    //一个古老的bug
    ~~~
  + 对于对象 除了函数都会显示object 
    ~~~
    typeof [] // 'object'
    typeof {} // 'object'
    typeof console.log // 'function'
    ~~~
## 四则运算符
  ~~~
  1 + '1' // '11'
  2 * '2' // 4
  [1, 2] + [2, 1] // '1,22,1'
  // [1, 2].toString() -> '1,2'
  // [2, 1].toString() -> '2,1'
  // '1,2' + '2,1' = '1,22,1'
  ~~~
  + 乘法运算符时 把string转num
  + 加法时 把number转string  
    
  * 特殊情况
  ~~~
  'a' + + 'b' // -> "aNaN"
  // 因为 + 'b' -> NaN
  // 你也许在一些代码中看到过 + '1' -> 1
  ~~~
